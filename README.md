### ZCreator
(Текущая версия приложения 6.1)

![Screen](https://mysoc126-svg.github.io/ZCreator.png)

Данная утилита предназначена для сборки (и разборки) ромов игр в формат ZMD/ZFC/ZSF/ZGB/ZPC/ZFB для игровых приставок:
 ZMD (сега или денди)
 ZFC (денди)
 ZSF (супер нинтендо)
 ZGB (гейм бой)
 ZPC (pc engine)
 ZFB (arcade)
 
Ромы игр могут быть в виде:
 *.bin;*.gen;*.md;*.smd (сега)
 *.nes;*.nfc;*.fds;*.fc (денди)
 *.fig;*.sfc;*.smc;*.swc (супер нинтендо)
 *.gb;*.gbc;*.gba (гейм бой)
 *.pce (pc engine)
 
Рекомендуется вместе с ромом игры добавлять и обложку в формате bmp/png/jpg.

Примечание: для последующей переупаковки ранее извлеченного рома, не забудьте сначала вытащить его из zip-архива.

Технически форматы ZMD/ZFC/ZSF/ZGB/ZPC/ZFB между собой совместимы. То есть, допустимо, например, ZFC файл переименовать в ZMD.

### PBitmap_AC
(Текущая версия приложения 2.3)

![Screen](https://mysoc126-svg.github.io/Pbitmap.png)

Программа для замены изображений фона игрового меню.

  Размеры изображений:
  - Главный фон (853х480)
  - Меню настроек (256х480)
  - Прочее (597х80)
  - Кнопки Yes No (217х37)
  - Курсор (смотри в Foldername.ini)

Поддержка прозрачности для кнопок Yes No и Курсора

### Foldername
(Текущая версия приложения 1.5)

![Screen](https://mysoc126-svg.github.io/Foldermane.png)

Полноценный графический редактор foldername.ini, который позволяет проверить/настроить навигацию по меню, редактировать цвета текста в категориях, положение и размеры обложки, курсора, названия папок и т.д. с полным графическим отображением вносимых изменений
